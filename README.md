Техническое задание для разработки консольного калькулятора на языке программирования C#
Цель работы: создать простое консольное приложение на языке C#, выполняющее базовые арифметическая операции с двумя числа. Интерфейс взаимодействия пользователя с программой осуществляется исключительно через терминал командной строки. Приложение должно обеспечивать выполнение операций:
1.	Сложение
2.	Вычитание 
3.	Умножение
4.	Деление 
5.	Возведение в степень
6.	Сравнить два числа:
a)	Переменная a больше переменной b
b)	Переменная a меньше переменной b
c)	Переменная a равно переменной b
Также программа должна включать проверку корректности вводимых данных, проверка на правильный выбор арифметической операции. 
Тест кейсы:
1.	Операция сложение: переменная a = 5, переменная b = 5. Результат a + b = 10
2.	Операция сложение: переменная a = -10, переменная b = -5. Результат a + b = -15
3.	Операция сложение: переменная a = -2, переменная b = 80. Результат a + b = 78
4.	Операция вычитание: переменная a = 5, переменная b = 5. Результат a - b = 0
5.	Операция вычитание: переменная a = -7, переменная b = -3. Результат a - b = -4
6.	Операция вычитание: переменная a = -5, переменная b = 5. Результат a - b = -10
7.	Операция умножение: переменная a = 6, переменная b = 7. Результат a * b = 42
8.	Операция умножение: переменная a = -4, переменная b = -4. Результат a * b = 16
9.	Операция умножение: переменная a = -10, переменная b = 7. Результат a * b = -70
10.	Операция деление: переменная a = 6, переменная b = 2. Результат a / b = 3
11.	Операция деление: переменная a = -10, переменная b = -2. Результат a / b = 5
12.	Операция деление: переменная a = -8, переменная b = 4. Результат a / b = -2
13.	Операция возведение в степень: переменная a = 4, переменная b = 2. Результат a ^ b = 16
14.	Операция возведение в степень: переменная a = 2, переменная b = 3. Результат a ^ b = 8
15.	Операция возведение в степень: переменная a = 0, переменная b = 4. Результат a ^ b = 0
16.	Операция сравнения двух чисел: переменная a = 10, переменная b = 2, c = “>”. Результат a > b = Истина
17.	Операция сравнения двух чисел: переменная a = 10, переменная b = 2, c = “<”. Результат a < b = Ложь
18.	Операция сравнения двух чисел: переменная a = 15, переменная b = 15, c = “=”. Результат a = b = Истина
